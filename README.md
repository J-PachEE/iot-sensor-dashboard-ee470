# IoT Sensor Dashboard and Networked Control Projects

**EE 470: Introduction to Internet-of-Things Coursework**

## Overview

This repository contains cleaned and reorganized Internet-of-Things coursework projects from EE 470. The projects focus on embedded device communication, sensor data collection, web-based control, PHP/MySQL backend integration, and networked IoT applications.

The purpose of this repository is to demonstrate hands-on experience with IoT system design, including embedded firmware, web interfaces, database logging, and remote device control.

## Course Context

EE 470: Introduction to Internet-of-Things introduced the design principles, components, infrastructure architectures, and networking protocols used to develop IoT systems. The course included hands-on projects involving embedded devices, sensors, web interfaces, and networked control applications.

## Topics Covered

* Embedded IoT firmware
* ESP-based device programming
* WiFi-based communication
* Sensor data collection
* PHP web interfaces
* MySQL database logging
* Web-based LED/RGB control
* HTTP-based device communication
* IoT dashboard development
* Event-triggered notifications

## Repository Structure


- firmware/       Embedded device firmware
- web/            PHP web interfaces and server-side scripts
- database/       SQL table structures and database notes
- docs/           Project documentation and notes
- images/         Screenshots, diagrams, and project visuals


## Project Areas

### Sensor Dashboard

Embedded devices collect sensor data and send readings to a web/database backend for logging and display.

### RGB / LED Web Control

Web-based control pages allow remote interaction with LED and RGB output devices through networked commands.

### Lux Monitoring

A light-level monitoring project that measures brightness and provides threshold-based feedback.

### HQTT / Networked Communication

IoT communication project involving embedded firmware and a PHP backend for networked device interaction.
